DDL (Data Definition Language)


Create (Database / Table)

Membuat Database / Tabel "CREATE DATABASE databasename; / CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, .... );"

Drop (Database / Table)

Menghapus Database / Tabel "DROP DATABASE databasename; / DROP DATABASE databasename;"

Alter (Database / Table)

Menyunting struktur Database / Table yang ada "ALTER TABLE table_name ADD column_name datatype;"


DML (Data Manipulation Language)


Insert (Table)

Menambahkan data baru ke dalam Table "INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);"
