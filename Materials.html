DDL (Data Definition Language)


Create (Database / Table)

Membuat Database / Tabel "CREATE DATABASE databasename; / CREATE TABLE table_name ( column1 datatype, column2 datatype, column3 datatype, .... );"

Drop (Database / Table)

Menghapus Database / Tabel "DROP DATABASE databasename; / DROP TABLE tablename;"

Alter (Table)

Menyunting struktur Table yang ada "ALTER TABLE table_name ADD column_name datatype;"


DML (Data Manipulation Language)


Insert (Table)

Menambahkan data baru ke dalam Table "INSERT INTO table_name (column1, column2, column3, ...) VALUES (value1, value2, value3, ...);"

Update (Table)

Memodifikasi data yang sudah ada dalam Table "UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;"

Delete (Table)

Menghapus kolom / data dari Table "DELETE FROM table_name WHERE condition;"

Select (Table)

Mengambil Data dari Table "SELECT CustomerName, City FROM Customers;"

Join (Table)

Menggabungkan 2 data atau lebih antar Table "SELECT Orders.OrderID, Customers.CustomerName, Orders.OrderDate FROM Orders INNER JOIN Customers ON Orders.CustomerID=Customers.CustomerID;"


Advanced


Order By

Use ORDER column_name BY ASC or DESC to sort data from the selected column
"Select * from bookshelf ORDER Author BY ASC;".

WHERE clause (Select, Delete, Update)

Used to extract datas that fulfilled specified condition
"Select * from bookshelf WHERE author = "Tian";".

And, Or, Not Operators

can be combined with WHERE clause.
use AND if both condition must be true.
use OR if only one condition is true.
use NOT if the condition is not TRUE
"Select * from bookshelf where Author = "Audrick" AND publisher = "Busur Setengah Lingkaran";"

Limit clause

used to specify a number of data to be shown "Select * from bookshelf WHERE author = "Tian" Limit 5;"

Min, Max

Use MIN() to show the smallest value in selected column.
Use MAX() to show the largest value in selected column
"Select MAX(id) from bookshelf;" "Select MIN(id) from bookshelf;"

SUM, COUNT, AVG

Use SUM to add up all the value of selected numeric column.
Use COUNT to count the numbers of rows that matches specified condition.
Use AVG to make an average value of a numeric column.
"Select COUNT(book_name) from bookshelf where publisher = "Wangsheng Funeral Parlor";"

LIKE

Used like WHERE clause to find certain data with specified pattern
"Select book_name from bookshelf WHERE book_name LIKE 'a%';"

(Meaning: show book name column from bookshelf table that starts with an "a")

Wildcard (%, _)

'%'' represent 0. 'br%' can find 'brick, broad, bread, break'.
'_' represent a single character. 'p_t' can find 'pet, pat, put, pot'
"select * from bookshelf WHERE author LIKE '_e%';"

(Meaning: show data from bookshelf table with an author name that has 'e' as the second letter of the name)


Group By

Used to show rows that has the same values.
Often used with SUM, AVG, COUNT, MIN, MAX
Use MAX() to show the largest value in selected column
"Select MAX(id) from bookshelf;" "Select MIN(id) from bookshelf;"

Primary Key

Primary Key is a unique key used to contain unique value and not NULL value
1 Table can only have 1 primary key however, Primary key can consist of multiple columns
"Create table customer_lists(Customer_id int PRIMARY KEY AUTO_INCREMENT, customer_name varchar(225), address varchar(225));"

(Note. AUTO INCREMENT is used to automatically generate unique value for each new row inserted).

Foreign Key

Used to establish link between two tables to a Primary Key
"Create table order_lists(Order_Id int AUTO_INCREMENT, Book_Amount int, Order_Date varchar(255), Customer_id int, FOREIGN KEY (Customer_id) REFERENCES customers_lists(Customer_id));"


Export & Import


BackUp / Export

Used to export your selected database into file.
Using Command prompt.
1.Type "exit"
2.The syntax would be "Mysqldump -u root -p your_database_name > file_path/file_name.sql; (For example: Mysqldump -u root -p Library > C:user/asus/documents/MyLibrary.sql)"
3.Type your password (If you don't have one, just press enter)

Restore / Import

Used to restore your database / Import your database from file.
Using Command prompt.
1.Create a database (any name)
2.use the database
3.The syntax would be "Source file_path/file_name.sqll; (For example: Source C:user/asus/documents/Mylibrary.sql)"
