DML (Data Manipulation Language)


Insert (Table)

Menambahkan data baru ke dalam Table

Update (Table)

Memodifikasi data yang sudah ada dalam Table

Delete (Table)

Menghapus kolom / data dari Table

Select (Table)

Mengambil Data dari Table

Join (Table)

Menggabungkan 2 data atau lebih antar Table
